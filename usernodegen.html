유저노드 파일 생성기

앞/뒤에 연결되는 노드와 필요한 기능을 설명하면, IRIS-Edge 의 사용자 정의 노드에서 선택해 실행할 user_node_xxx.py 스크립트 한 개를 생성합니다.

이 도구는 입력한 Claude API 키로 사용자의 Claude 를 직접 호출합니다. 키는 이 브라우저(localStorage)에만 저장되며 서버로 전송되지 않습니다.
유저노드 스크립트는 run(node_data_type_list, data, node_data, log_info) 함수를 정의하며, (result, node_data) 튜플을 반환합니다. data 는 앞 노드의 node_id 를 key 로 하는 dict 입니다. 생성된 코드에는 CAMERA_GRAB_NODE_ID 같은 자리표시자가 들어가며, 실제 노드 ID 를 복사-붙여넣기 하면 됩니다.
