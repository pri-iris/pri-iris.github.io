IRIS-Edge 문서 센터

IRIS-Edge 와 차세대 IRIS-Edge 2.0 의 호환 장치, 통신 프로토콜, API, 그리고 워크플로우 노드를 한곳에서 확인하세요.

IRIS-Edge 는 카메라·로봇·PLC·AI 모델을 하나의 비주얼 워크플로우로 묶어 현장 자동화·검사 시퀀스를 구성하는 엣지 플랫폼입니다. IRIS-Edge 2.0 은 동일한 자동화 로직을 GUI 에 종속되지 않는 gRPC / REST API 로 노출하여, 어떤 프론트엔드에서도 같은 하드웨어 제어와 워크플로우 실행을 구동할 수 있습니다.
